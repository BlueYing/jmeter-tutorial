# 结束语

哇，你竟然看到**最后**了，看来你真的很想把性能测试做好。那就再送你点`建议`

- 测试过程中，如果出现脚本`调试不通或者难以调试`时，记得`及时`请教开发人员

- 针对有些可重复利用的数据，尽量要`好好保存`。减少不必要的`重复劳动`

- 不要过于依赖`工具`，有时电脑自身的功能就可以满足你。如:

  - Windows的`资源监视器`/Unix中的`TOP`就提供了常用的性能数据

  - 工具统计的数据会`额外`占用你的机器资源


-------------------------------------------------------


最后，希望你性能测试**一切顺利**！
